# Blitzkrieg Tracker

Blitzkrieg Tracker is a mod that simplifies using the **<cy>Blitzkrieg training tactic</c>** in Geometry Dash.

Instead of managing spreadsheets or manually tracking runs, the mod automatically records your **<cg>progress directly during gameplay</c>**.

# Quick Tutorial

Getting started with Blitzkrieg is simple:

1. Create a **<cg>copy of the level</c>** with multiple **<cy>start positions</c>** *(around **<cy>12 start positions</c>** is usually ideal).*  
   You don’t have to create the copy yourself — you can also **<cg>find startpos copies made by other players</c>** in the level search.
2. Open the mod menu and **<cg>create a profile</c>** for that level.
3. The mod will automatically **<cg>generate stages</c>** based on the start positions.
4. Start playing — the mod will **<cg>track runs, attempts, and playtime automatically</c>**.

You can also bind the same profile to **multiple levels**, such as a **startpos copy** and the **original level**, allowing you to track runs from **0%** while still practicing with start positions.

# Key Features

* **<cg>Profile System</c>** — create profiles for specific levels in just a few clicks.
* **<cg>Automatic Stage Generation</c>** — stages are generated automatically from the start positions in your level copy.
* **<cg>Run Tracking</c>** — the mod detects and records completed runs during gameplay.
* **<cg>Detailed Statistics</c>** — track attempts, completions, best runs, first runs, and time played.
* **<cg>Stage Progression</c>** — see which runs are required for the current stage and track overall progress.
* **<cg>Multiple Level Binding</c>** — a single profile can be linked to multiple levels (for example, a startpos copy and the original level).
* **<cy>Noclip Detection</c>** — runs are not counted if the mod detects a noclip death.
* **<cg>Notifications</c>** — receive notifications when runs or stages are completed.

# Statistics Tracked

Each run records detailed information:

* **<cg>Time Played</c>** — total time spent attempting the run  
* **<cg>First Run</c>** — first recorded progress  
* **<cg>Best Run</c>** — best result achieved  
* **<cg>Completions</c>** — number of successful run completions  
* **<cg>Completed In</c>** — number of attempts required for the first completion  
* **<cg>Attempts</c>** — total attempts on the run  

Stage statistics also include **<cg>total attempts and total playtime</c>**.

# Additional Features

* **<cg>Stage Browser</c>** — view past and upcoming stages and their required runs.
* **<cg>Pinned Profiles</c>** — pin important profiles to keep them at the top of the list.
* **<cg>Custom SFX Support</c>** — play custom sounds when runs or stages are completed.
* **<cy>Practice Mode Ignore Option</c>** — optionally ignore progress made while using cheats in practice mode (for training difficult parts).

# Who Is This For

* Players practicing **<cg>extreme demons</c>**
* Players using the **<cy>Blitzkrieg method</c>** to improve consistency
* Anyone who wants **<cg>structured training and automatic progress tracking</c>**
