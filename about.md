# Blitzkrieg Tracker

**Blitzkrieg Tracker** is a mod that simplifies using the **<cy>Blitzkrieg training method</c>** in Geometry Dash.

Instead of manually tracking runs or using spreadsheets, the mod automatically records your **<cg>progress directly during gameplay</c>**.

The Blitzkrieg training method was originally **invented by [Rudder](https://www.youtube.com/@TheRealRudder)**.
This mod is inspired by that approach and automates the tracking process.

# Quick Tutorial

Getting started is simple:

1. Create a **<cg>copy of a level</c>** with multiple **<cy>start positions</c>**
   *(~12 start positions is usually ideal)*
   — or find an existing startpos copy in the level search.
2. Open the mod menu and **<cg>create a profile</c>** for the level.
3. The mod will automatically **<cg>generate stages</c>** based on your start positions.
4. Start playing — runs, attempts, and playtime are tracked automatically.

You can also bind a profile to **multiple levels** (e.g. a startpos copy and the original level), allowing you to track progress from **0%** while still practicing efficiently.

# Key Features

- **<cg>Profile System</c>** — create and manage level profiles in seconds
- **<cg>Automatic Stage Generation</c>** — stages are generated from your start positions
- **<cg>Run Tracking</c>** — runs are detected and recorded automatically
- **<cg>Detailed Statistics</c>** — attempts, completions, best runs, playtime, and more
- **<cg>Stage Progression</c>** — track required runs and overall progress
- **<cg>Multi-Level Binding</c>** — link one profile to multiple levels
- **<cy>Noclip Detection</c>** — invalid runs are ignored automatically
- **<cg>Notifications</c>** — get notified about completed runs and stages

# Statistics Tracked

Each run records:

- **<cg>Time Played</c>** — total time spent on the run
- **<cg>First Run</c>** — first recorded progress
- **<cg>Best Run</c>** — best achieved progress
- **<cg>Completions</c>** — number of successful completions
- **<cg>Completed In</c>** — attempts required for the first completion
- **<cg>Attempts</c>** — total attempts

Stage statistics also include **total attempts and total playtime**.

# Additional Features

- **<cg>Stage Browser</c>** — browse past and upcoming stages and see required runs
- **<cg>Stage Graph</c>** — visualize your progress across all stages, including attempts and playtime trends
- **<cg>High-Quality Start Positions</c>** — access and use curated startpos copies for more efficient training
- **<cg>Custom SFX</c>** — play custom sounds on run or stage completion

# Who Is This For

- Players practicing **extreme demons**
- Users of the **Blitzkrieg method**
- Anyone who wants **structured training and automatic progress tracking**
